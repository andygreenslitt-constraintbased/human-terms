## 2. Whole-Document Human Lyric

This repo is not saying "this is what ancient Egypt did."

It is saying:

```text
If we model a proposed build pathway as an engineering constraint system,
which parts survive the gates,
which parts fail,
which parts remain conditional,
and which claims must be blocked?
```

The strongest safe translation is:

```text
This is an engineering screen, not historical proof.
```

The model reaches a completed-build pathway under stated assumptions without a required simulated gate failure. That is useful. But it remains inside the simulation boundary. It does not become archaeology, historical fact, or proof of hydraulic construction.

## 3. Whole-Document Technical Notes

Source body:

- `GIZA_RELEASE_CLAIMS.md`: allowed, conditional, and blocked public claims.
- `GIZA_SIMULATION_RESULTS_SUMMARY.md`: final simulation status, surviving system, non-surviving claims.
