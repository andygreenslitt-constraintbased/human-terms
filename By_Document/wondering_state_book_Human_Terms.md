## 2. Whole-Document Human Lyric

This book is about what to do when you can tell a structure matters before you can fully prove, build, or explain it.

The central model is:

```text
ZERO      = no legible necessity yet
WONDERING = necessity has become visible, but full implementation is not known
KNOWING   = lawful application after the necessity is legible enough to act
```

In human terms, the book says:

```text
Do not pretend to know too early.
Do not dismiss the signal just because it is not fully built yet.
Stay in the wondering state long enough to find the boundary of what must be true.
```

The book uses systems language to explain how things fail: systems begin at a fixed point, movement creates decay, transfers must be clean, confusion is a signal problem, the weakest sustainable component sets the ceiling, hidden load can lower capacity, repair must preserve the original reason for the system, and authority should live where material contribution and complexity handling match.

The readable version should preserve the three-state model without upgrading it into proof of cognition, psychology, neuroscience, or a universal theory of mind.

## 4. Locked Document Meaning

The locked meaning is the three-state model:

```text
ZERO:
The necessity is not yet legible.

WONDERING:
The necessity is visible before the complete implementation is known.

```

The book's practical discipline is to remain honest about which state you are in. It blocks premature certainty while also protecting early structural signal from being dismissed merely because it is not fully implemented yet.
