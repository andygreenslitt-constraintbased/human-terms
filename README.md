# Human Terms

Plain-language Technical Score companions for the constraint-navigation framework.

Each companion translates a dense source document using the [Technical Score Translator Output Directive](TECHNICAL_SCORE_TRANSLATOR_OUTPUT_DIRECTIVE.md).

## Format

Every companion uses four layers:

1. **Human Lyric** — plain-language meaning
2. **Technical Notes** — source structure, faithful paraphrase, and exact quote where available
3. **Locked Meaning** — the invariant that must survive any translation
4. **Audit Layer** — dependencies, assumptions, failure risks, confidence, and translation status

## Non-claims

Readable language may explain the source. Readable language may not replace, distort, erase, or overclaim the source.

These companions do not claim proof, validation, correctness, production readiness, public readiness, execution authority, expert replacement, or domain certification.
